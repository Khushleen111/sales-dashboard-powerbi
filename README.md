# Sales Performance Dashboard – Power BI

## Overview
This project analyzes sales performance using an interactive Power BI dashboard.

## Tools Used
- Power BI
- SQL dataset

## Key Insights
- Revenue generated by each Zone and Year
- Top 5 customers and products based on sales
- Sales quantity by market
- Sales performance trends across different regions
